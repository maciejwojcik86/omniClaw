## MODIFIED Requirements

### Requirement: View Node Budgets
The system SHALL provide an operational tool or API to view the current spend, daily allowance, available virtual key metadata, and aggregated session usage stats for any agent.

#### Scenario: Operator views budget
- **WHEN** an operator runs the cross-comparison skill/tool
- **THEN** the system returns the exact spend, limits, and the aggregated native token usage per agent
